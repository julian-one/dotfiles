# Dotfiles

## Usage

```bash
git clone https://github.com/julian-one/dotfiles.git ~/dotfiles
cd ~/dotfiles
make all
```

## Commands

```bash
make all              # Complete setup
make install          # Install packages
make stow             # Create symlinks
make clean            # Remove symlinks
make help             # Show all commands
```

## What's Included

- **Neovim** - LSP, treesitter, completion
- **Bash** - Simple prompt with git branch
- **Ghostty** - Terminal with Comic Code font
- **Git** - Profile configurations
